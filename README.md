# react-next-bolierplate

react ssr bolierplate, use antd, typescript, koa2...

https://github.com/flasco/react-next-bolierplate

## env-require

- node 8+
- yarn

## how-to-run

```bash
cd <project root dir>
yarn # install dependencies
yarn server # start server
```
